About the Projection data

Image data detailing axonal projections labeled by rAAV tracers and visualized using serial two-photon tomography

  • Specific anatomic regions throughout the brain
  • Numerous defined neuronal populations
  • Image stacks obtained by serial two-photon tomography

About the Allen Mouse Brain Connectivity Atlas

A high-resolution map of neural connections in the mouse brain, built on an array of transgenic mice genetically engineered to target specific cell types

Key features

  • Projection Mapping image data detailing axonal projections labeled by viral tracers
  • BDA vs. AAV Comparison dataset for comparing conventional and viral tracing methods
  • Transgenic Characterization data detailing transgene expression in Cre or other driver lines
  • Anatomic Reference
